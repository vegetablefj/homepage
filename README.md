# Academic Homepage

Minimal academic homepage. Built under the assistance of DeepSeek.

## Pages

- **Home**: Personal introduction and contact
- **Research**: Research interests and projects  
- **Publications**: Academic publications
- **Preprints**: Preprints and working papers
- **Notes**: Lecture and research notes
- **Miscellaneous**: Useful links and resources

## Technology

- Pure HTML/CSS
- Responsive design
- GitHub Pages compatible

## Customization

Edit HTML files to update content. Add PDFs to `pdfs/` directory and link to them.
